
# Java Fundamentals Course — AgroTrack

Curso introductorio de Java para estudiantes de secundaria (12–17 años),
enfocado en los fundamentos de la Programación Orientada a Objetos (POO).
¡No requiere instalación de ninguna herramienta!

## Contenido
Accede a "course-plan.md" donde podrás ver el curso completo con las lecciones.

## Configuración

No necesitas instalar nada. Para ejecutar el código de este repositorio,
abre cualquiera de estos editores online desde tu navegador:

- [Replit](https://replit.com)
- [JDoodle](https://www.jdoodle.com/online-java-compiler)
- [OnlineGDB](https://www.onlinegdb.com/online_java_compiler)

Simplemente copia el código de la carpeta que necesites y pégalo en el editor.

## Requisitos

- Navegador web (Chrome, Firefox, Safari o Edge)
- Sin descargas, sin instalaciones, sin registros obligatorios
