# Script - Lección 2: Tipos de Datos en Java

## ¿Qué es una variable?

Una variable es un espacio en memoria donde se almacena información.

Podemos imaginar una variable como una caja etiquetada que guarda un dato.

Ejemplos:

- Edad
- Nombre
- Precio

## ¿Qué son los tipos de datos?

Los tipos de datos indican qué clase de información puede almacenar una variable.

Java necesita conocer el tipo de dato para guardar correctamente la información
y utilizar la memoria de forma eficiente.

## Memoria y optimización

Cada tipo de dato ocupa una cantidad diferente de memoria.

| Tipo | Tamaño |
|--------|--------|
| byte | 1 byte |
| short | 2 bytes |
| int | 4 bytes |
| long | 8 bytes |
| float | 4 bytes |
| double | 8 bytes |
| char | 2 bytes |

Utilizar correctamente los tipos de datos ayuda a optimizar la memoria y mejorar
el rendimiento de las aplicaciones.

## Tipo int

Almacena números enteros.

```java
int edad = 20;
```

## Tipo double

Almacena números decimales.

```java
double altura = 1.75;
```

## Tipo char

Almacena un único carácter.

```java
char inicial = 'J';
```

## Tipo String

Almacena texto.

```java
String nombre = "Juan";
```

## Tipo boolean

Almacena valores lógicos.

```java
boolean aprobado = true;
```

## Ejemplo práctico

```java
int edad = 20;
double altura = 1.75;
char inicial = 'J';
String nombre = "Juan";
boolean estudiante = true;
```

## Conclusión

Los tipos de datos permiten almacenar información de forma correcta y eficiente.

Los más utilizados son int, double, char, String y boolean.
