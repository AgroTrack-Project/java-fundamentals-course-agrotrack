# Script - Lección 1: ¿Qué es la Programación y qué es Java?

## ¿Qué es la programación?

La programación es el proceso de crear instrucciones para que una computadora
realice determinadas tareas.

Las computadoras no piensan ni toman decisiones por sí solas. Necesitan que una
persona les indique exactamente qué hacer mediante una serie de instrucciones.

La programación no solo consiste en escribir código. Antes de programar, es
necesario analizar un problema, pensar una solución y luego convertir esa
solución en instrucciones que la computadora pueda entender.

Por esta razón, la programación también ayuda a desarrollar habilidades como
el pensamiento lógico, la creatividad y la resolución de problemas.

## Ejemplo cotidiano

Imaginemos que queremos preparar una taza de café.

Podríamos seguir los siguientes pasos:

1. Tomar una taza.
2. Agregar café.
3. Agregar agua caliente.
4. Mezclar.

Estos pasos forman una secuencia de instrucciones. De manera similar, un
programa es una serie de instrucciones que la computadora ejecuta para
obtener un resultado.

## ¿Para qué sirve la programación?

La programación nos permite desarrollar aplicaciones y sistemas que utilizamos
diariamente.

Algunos ejemplos son:

- Redes sociales.
- Aplicaciones móviles.
- Videojuegos.
- Sistemas bancarios.
- Plataformas educativas.
- Tiendas virtuales.

Actualmente la programación está presente en casi todos los sectores,
incluyendo educación, medicina, transporte, comercio, entretenimiento y
comunicaciones.

Por ejemplo, cuando realizamos una compra por internet o utilizamos una
aplicación para pedir comida, detrás de esas acciones existe un sistema
desarrollado por programadores.

## ¿Qué es un lenguaje de programación?

Un lenguaje de programación es un conjunto de reglas y palabras que permiten
comunicarnos con una computadora.

Los lenguajes de programación sirven para escribir instrucciones que luego
serán interpretadas y ejecutadas por la computadora.

Existen muchos lenguajes de programación, entre ellos:

- Python
- Java
- C++
- JavaScript
- C#

Aunque existen muchos lenguajes de programación, todos tienen el mismo
objetivo: permitir que una persona pueda dar instrucciones a una computadora.

La diferencia es que algunos lenguajes son más sencillos para ciertas tareas
y otros son más adecuados para proyectos grandes o complejos.

## ¿Qué es Java?

Java es un lenguaje de programación creado en 1995 por Sun Microsystems.

Actualmente pertenece a Oracle y es uno de los lenguajes más utilizados en
el desarrollo de software.

Java destaca por ser seguro, robusto y multiplataforma.

Además, es uno de los lenguajes más enseñados en universidades e institutos
porque permite aprender conceptos fundamentales de programación.

## Características principales de Java

### Orientado a objetos

Java utiliza el paradigma de programación orientada a objetos.

Este paradigma permite organizar mejor el código, reutilizar componentes y
facilitar el desarrollo y mantenimiento de aplicaciones grandes.

### Multiplataforma

Una de las características más conocidas de Java es que el mismo programa
puede ejecutarse en diferentes sistemas operativos como Windows, Linux y MacOS.

Por esta razón se le asocia frecuentemente con la frase:

> Write Once, Run Anywhere

Que significa: escribir una vez y ejecutar en cualquier lugar.

### Amplio uso empresarial

Muchas empresas utilizan Java para desarrollar sistemas bancarios,
aplicaciones corporativas y servicios web.

Su estabilidad y rendimiento hacen que sea una opción muy utilizada en el
sector profesional.

## ¿Dónde se utiliza Java?

Java se utiliza en diversos campos, entre ellos:

- Aplicaciones empresariales.
- Aplicaciones Android.
- Sistemas financieros.
- Plataformas web.
- Sistemas de gestión empresarial.

Muchas aplicaciones y servicios que utilizamos diariamente cuentan con
componentes desarrollados en Java.

## JDoodle

Para nuestras prácticas utilizaremos JDoodle.

JDoodle es un entorno de desarrollo en línea que permite escribir, ejecutar
y probar código Java directamente desde el navegador.

Su principal ventaja es que no requiere instalar programas adicionales,
facilitando el aprendizaje y la realización de ejercicios.

## Ejemplo sencillo en JDoodle

```java
public class Main {
    public static void main(String[] args) {

        System.out.println("INTRODUCCION A JAVA");
        System.out.println("La programacion permite resolver problemas.");
        System.out.println("Java es un lenguaje de programacion.");
        System.out.println("Se utiliza en aplicaciones web, moviles y empresariales.");

    }
}
```

Este programa muestra mensajes en pantalla utilizando la instrucción
`System.out.println()`. Cada línea representa una instrucción que la
computadora ejecutará en orden.

Aunque es un programa sencillo, nos permite observar cómo Java sigue una
secuencia de instrucciones para producir un resultado.

## Conclusión

La programación consiste en crear instrucciones para resolver problemas
mediante una computadora.

Para ello utilizamos lenguajes de programación, y uno de los más importantes
es Java, debido a su seguridad, versatilidad y amplio uso en la industria
del software.
